# 暦 -KOYOMI- 六曜カレンダー（iPhone用PWA）

六曜・干支・日本の祝日・二十四節気・月齢・雑節・メモ機能を備えた、
iPhoneのホーム画面に追加して使えるカレンダーアプリです。

## 特徴・できること

- 📅 六曜（大安・友引・仏滅など）
- 🐍 干支（日の干支・年の干支）
- 🇯🇵 日本の祝日（振替休日・国民の休日も自動計算）
- 🌕 二十四節気
- 🌑 月齢（月の満ち欠け）
- 🌸 雑節（節分・彼岸・土用・八十八夜・入梅・半夏生・二百十日など）
- 📌 日付ごとのメモ
- 🔔 通知（下記「通知について」を参照）
- ☁️ バックアップの書き出し・読み込み（下記「バックアップについて」を参照）
- 🎨 ライト／ダークモード（自動・手動切り替え）

六曜・旧暦・二十四節気・月齢はすべて天文計算（太陽と月の位置の計算）による
自動算出です。テストの結果、2026年の祝日は国立天文台発表の内容と完全に一致することを確認しています。
ただし、まれに旧暦・六曜が専門書と1日ずれることがあります。

## 【超重要】プログラミング初心者の方へ：使うための3ステップ

このアプリは「ファイル一式」なので、iPhoneで使うには
**①インターネット上のどこかに置く → ②SafariでURLを開く → ③ホーム画面に追加する**
という3ステップが必要です（普通のアプリのようにApp Storeからは入れられません）。

### ① どこかにアップロードする（おすすめ：GitHub Pages）

一番かんたんで無料なのは GitHub Pages です。

1. https://github.com でアカウントを作る（すでにお持ちならそのまま）
2. 新しいリポジトリを作成する（例：`rokuyou-calendar`）
3. このフォルダの中身（index.html, manifest.json, sw.js, css, js, icons）を
   すべてそのリポジトリにアップロードする（GitHubのWeb画面から
   「Add file」→「Upload files」でドラッグ＆ドロップでもOK）
4. リポジトリの Settings → Pages で、Branch を `main`（フォルダは `/root`）にして保存
5. しばらくすると `https://あなたのユーザー名.github.io/rokuyou-calendar/` で
   アクセスできるようになります

他にも Netlify や Cloudflare Pages にドラッグ＆ドロップでアップロードする方法もあります。
（PWAとして正しく動く＝ホーム画面に追加してオフラインでも開けるようにするには、
`https://` で始まるアドレスで公開する必要があります。）

### ② iPhoneのSafariでURLを開く

上記で公開したURLを、iPhoneの **Safari**（Chromeなど他のブラウザは不可）で開きます。

### ③ ホーム画面に追加する

Safari下部の共有ボタン（□に↑のアイコン）→「ホーム画面に追加」を選ぶと、
ホーム画面にアプリのアイコンが追加され、以降はアプリのように起動できます。

## 通知について（正直な注意点です）

iPhoneのSafari／ホーム画面PWAの通知には仕様上の制限があります：

- アプリを開いている間だけ届く「その場限りの通知」は実装済みです（設定タブでON）
- アプリを完全に閉じている間も届く「バックグラウンド通知」を実現するには、
  本来は通知配信専用のサーバー（プッシュサーバー）が必要で、
  このような単体のファイル一式だけでは実現できません

そのため、確実にリマインダーを届けたい日は、日付詳細画面の
「カレンダーに書き出す」ボタンで `.ics` ファイルを作成し、
iPhone標準の「カレンダー」アプリに読み込んでください。
標準カレンダーの通知は、アプリを閉じていても確実に届きます。

## バックアップについて（正直な注意点です）

メモや設定は、あなたのiPhoneの中（ブラウザのストレージ）に保存されます。
Apple公式の「iCloud」に自動で同期する機能は、無料のApple Developer登録なしでは
技術的に作れないため、代わりに次の方法を用意しています：

1. 設定タブの「書き出す」でメモ・設定を1つのファイル（.json）として保存
2. iPhoneの「ファイル」アプリで、そのファイルを iCloud Drive のフォルダに移動
3. 別の端末や機種変更後は、同じアプリを開いて「読み込み」からそのファイルを選ぶと復元されます

自動同期ではなく「手動でのバックアップ」ですが、データが消えてしまう心配は防げます。

## フォルダの中身

```
rokuyou-calendar/
├── index.html      … 画面本体
├── manifest.json    … PWAの設定（アプリ名・アイコンなど）
├── sw.js             … オフラインで開けるようにする仕組み
├── css/style.css     … デザイン
├── js/
│   ├── astro.js       … 太陽・月の位置を計算する土台
│   ├── lunar.js        … 二十四節気・旧暦・六曜の計算
│   ├── kanshi.js        … 干支（日・年）の計算
│   ├── zassetsu.js       … 雑節（彼岸・土用など）の計算
│   ├── holidays.js        … 日本の祝日の計算
│   └── app.js               … 画面の動き（カレンダー描画・メモ・通知・設定）
└── icons/            … アプリアイコン
```

## 動作確認環境

Safari（iOS）を想定して作っています。PCのChrome／Edge／Safariでも
レイアウト確認はできますが、通知やホーム画面追加はiPhoneのSafariでお試しください。
